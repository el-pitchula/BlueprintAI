# Blueprint AI

Blueprint AI is a multimodal generative AI project developed for the course **Generative Artificial Intelligence and Multimodal Models**.

The project explores parameter-efficient fine-tuning of Stable Diffusion using **LoRA (Low-Rank Adaptation)** to generate images in the style of **scientific technical illustrations**, inspired by engineering blueprints, technical manuals and scientific documentation.

The final application combines text, image and audio generation through an interactive Gradio interface deployed on Hugging Face Spaces.

---

## Features

- Stable Diffusion fine-tuning using LoRA
- Scientific technical illustration style
- Prompt-based image generation
- Multimodal pipeline
- Gradio web interface
- Hugging Face Spaces deployment
- CLIPScore and human evaluation
